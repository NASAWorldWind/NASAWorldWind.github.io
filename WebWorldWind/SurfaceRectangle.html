Class: SurfaceRectangle

SurfaceRectangle(center, width, height, heading, attributes)

Represents a rectangle draped over the terrain surface.

SurfaceRectangle uses the following attributes from its associated shape attributes bundle:

  • Draw interior
  • Draw outline
  • Interior color
  • Outline color
  • Outline width
  • Outline stipple factor
  • Outline stipple pattern
