JCenter hosting for Maven and Gradle Projects

The World Wind Android library is available from JCenter for projects that use the Maven or Gradle build management systems. Use the group id, artifact id, and versions listed below to easily integrate World Wind Android into your project. For an example of integrating World Wind Android with your project, see Step 3 of the Build Setup here.

  • Group Id: gov.nasa.worldwind.android
  • Artifact Id: worldwind
  • Version: undefined

Development builds of the library are also available. You must add the Open Source Software JFrog (OJO) repository to properly resolve the development artifacts. The development build uses the same Group and Artifact Id as the release. An example for Gradle is provided below.

  • Adding the OJO repository:
    •     repositories {
        maven {
            url "https://oss.jfrog.org/artifactory/oss-snapshot-local"
        }
    }
  • Latest SNAPSHOT Version: undefined
